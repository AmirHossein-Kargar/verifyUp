# راهنمای حل مشکل Bottom Navigation

## مشکل شما چیست؟

شما یک bottom navigation با کلاس‌های زیر می‌بینید:

```html
bg-neutral-primary-soft, tooltip, rounded-full, bottom-4
```

این کلاس‌ها **از کد ما نیستند**! این از یک افزونه مرورگر است.

## Bottom Navigation واقعی ما

کد ما از این کلاس‌ها استفاده می‌کند:

```html
bg-white dark:bg-gray-800 border-t border-gray-200 dark:border-gray-700
```

و در `bottom: 0` قرار دارد (نه `bottom-4`).

## چرا این اتفاق افتاده؟

یک افزونه مرورگر (احتماالً یکی از این‌ها):

- افزونه ترجمه (مثل Google Translate)
- افزونه دسترسی‌پذیری
- افزونه تم یا ظاهر
- افزونه‌های مدیریت تب
- افزونه‌های شبکه اجتماعی

این افزونه‌ها HTML اضافی به صفحه شما تزریق می‌کنند.

## راه حل‌ها

### 1. غیرفعال کردن افزونه‌ها (توصیه می‌شود)

1. به تنظیمات مرورگر بروید
2. بخش Extensions/افزونه‌ها را باز کنید
3. همه افزونه‌ها را غیرفعال کنید
4. صفحه را رفرش کنید

### 2. استفاده از حالت ناشناس (Incognito)

1. پنجره ناشناس باز کنید (Ctrl+Shift+N در Chrome)
2. به آدرس localhost بروید
3. bottom navigation ما را خواهید دید

### 3. استفاده از مرورگر دیگر

اگر مشکل ادامه دارد، از مرورگر دیگری استفاده کنید.

## صفحات تست

برای تست bottom navigation ما:

1. `/diagnostic` - صفحه تشخیص مشکل
2. `/test-nav` - صفحه تست ساده
3. `/dashboard` - داشبورد واقعی

## تایید اینکه مشکل حل شده

وقتی مشکل حل شد، باید:

✅ Bottom navigation در `bottom: 0` باشد (چسبیده به پایین)
✅ بدون گوشه‌های گرد (rounded-full)
✅ با border-top (خط بالایی)
✅ تمام عرض صفحه
✅ 4 آیتم: داشبورد، سفارشات، خدمات، پروفایل

## وضعیت فعلی کد

✅ Header برای صفحات dashboard و admin حذف شده
✅ Bottom navigation در `bottom: 0` قرار دارد
✅ تمام صفحات `pb-20` دارند (فضای پایین برای bottom nav)
✅ Responsive و تمام عرض است
✅ DashboardSkeleton هم آپدیت شده

## کد ما کاملاً صحیح است!

مشکل از افزونه مرورگر است، نه از کد ما. لطفاً افزونه‌ها را غیرفعال کنید.
